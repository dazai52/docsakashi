    List Server Streaming Akashinime¶

    Di web akashinime disarankan streaming daripada download karena tidak ada iklan.

    Server Resolusi Iklan
    Server Fast 360p&720p tidak ada
    Server 1 Multi Quality 360p-1080p tidak ada
    Server 2 Multi Quality 360p-1080p tidak ada
    Server 3 Ads 360p-1080p ada
    Server VIP 240p-1080p tidak ada
    Server Backup 144p-1080p tidak ada
    Server MG 360p-720p 360p-720p ada
    Server MP4upload 360p-720p 360p-720p ada

    Informasi Server Streaming¶

    • Server Fast, Server 1 dan 2 multi Quality, Server 3 Ads, Server VIP, Server Backup Support Multi Resolusi.
    • Server Fast merupakan Server tercepat dan merupakan Server Default Akashinime.
    • Server VIP / ADS / backup hanya beberapa saja yang bisa di play dikarenakan Copyright.
    • Server Fast hanya tersedia di anime series ( anime dengan durasi 24 Menit an ).
    • Jika Server 1 / 2 Multi Quality Lemot (biasanya terjadi di indihome). silahkan gunakan aplikasi 1.1.1.1 / VPN.

    Error Saat Streaming¶

    Pastikan sudah menggunakan browser Google Chrome versi terbaru

    • Server Streaming tidak bisa di play biasanya karena video rusak ( terhapus kena copyright ).
    • Server 1 Multi Quality / Server 2 Multi Quality terkadang mengalami bug loading.

    cara mengatasi bug tersebut cukup pindah dari server 1 ke server 2 dan kembali lagi ke server 1, begitu juga sebaliknya.
